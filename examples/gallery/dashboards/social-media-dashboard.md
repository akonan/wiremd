**Style Variations:**
- `sketch` - Quick social dashboard wireframe
- `clean` - Professional social media tool
- `wireframe` - Content structure focus
- `tailwind` - Modern vibrant interface
- `material` - Google-style social hub
- `brutal` - Bold engagement dashboard
- `none` - Semantic HTML for frameworks

**Generate this example:**
```bash
wiremd social-media-dashboard.md --style sketch
wiremd social-media-dashboard.md --style clean -o social-clean.html
wiremd social-media-dashboard.md --style tailwind -o social-tailwind.html
```
