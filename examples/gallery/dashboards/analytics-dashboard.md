**Style Variations:**
- `sketch` - Quick dashboard prototype
- `clean` - Modern analytics interface
- `wireframe` - Data structure focus
- `tailwind` - Contemporary SaaS dashboard
- `material` - Google Analytics aesthetic
- `brutal` - Bold data visualization
- `none` - Semantic HTML for charting libs

**Generate this example:**
```bash
wiremd analytics-dashboard.md --style sketch
wiremd analytics-dashboard.md --style material -o analytics-material.html
wiremd analytics-dashboard.md --style clean -o analytics-clean.html
```
