**Style Variations:**
- `sketch` - Quick e-commerce prototype
- `clean` - Professional shop dashboard

- `tailwind` - Modern retail interface
- `material` - Google Merchant style
- `brutal` - Bold commerce UI
- `none` - Semantic HTML for custom styling

**Generate this example:**
```bash
wiremd ecommerce-dashboard.md --style sketch

wiremd ecommerce-dashboard.md --style tailwind -o ecommerce-tailwind.html
```
